what is a Component?


Component is simple javascript class

every class will be prefixed with Export


Export make your class available every where so that that class you can use it



In class you can declare Variable,Methods,constructor etc..


class  will be storngly type


to make any class as Component what you need to do it?

@Component() Decorator on above your class
