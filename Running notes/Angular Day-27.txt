Creates an AbstractControl from a user-specified configuration.

The FormBuilder provides syntactic sugar that shortens creating instances of a FormControl, FormGroup, or FormArray. It reduces the amount of boilerplate needed to build complex forms.

studentForm.get('firstname').hasError(''Required")

What is FormBuilder?
The FormBuilder provides syntactic sugar that shortens creating instances of a FormControl
it is another of writing reactive form code
its like a key and value pair
here in UI you  will see formControlName

import :

import {FormBuilder, AbstractControl, FormGroup, ValidationErrors, Validators, ReactiveFormsModule } from '@angular/forms';

ts Code:

 

  studentForm: FormGroup;
  constructor(private fbuilder:FormBuilder)
  {
this.studentForm=this.fbuilder.group(
  {
    firstname:['',[Validators.required]],
    lastname:'',
    username:['',[Validators.required]],
    city:'',
    state:'',
    zipcode:'',
    IsAgreed:false

  })
  }

 
    OnSubmit()
{
  console.log(this.studentForm);
}
